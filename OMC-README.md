# Oh-My-ClaudeCode (OMC) 완전 초보자 가이드

> "Claude Code를 배우지 마세요. 그냥 OMC를 쓰세요."

OMC는 Claude Code를 훨씬 강력하게 만들어주는 플러그인입니다.
여러 AI 에이전트가 동시에 협력해서 복잡한 작업도 자동으로 처리해줍니다.

---

## OMC가 뭔가요?

Claude Code는 AI 비서 하나가 일하는 것이라면,
OMC는 **AI 팀 전체**가 나눠서 일하는 것입니다.

예를 들어 "앱 만들어줘"라고 하면:
- 한 AI는 설계를 담당
- 한 AI는 코드 작성 담당
- 한 AI는 검토 담당
- 한 AI는 버그 수정 담당

이 모든 게 자동으로 돌아갑니다.

---

## 기본 사용법 (이것만 알면 됩니다)

### 방법 1: 그냥 말하기 (가장 쉬운 방법)

Claude Code 채팅창에 하고 싶은 것을 그냥 한국어로 적으면 됩니다.

```
할 일 목록 앱 만들어줘
```

```
내 코드에서 버그 찾아서 고쳐줘
```

```
이 코드가 뭐 하는 건지 설명해줘
```

### 방법 2: autopilot (자동 조종)

복잡한 걸 처음부터 끝까지 혼자 다 해줍니다.

```
autopilot: 메모장 웹사이트 만들어줘
```

앞에 `autopilot:` 만 붙이면 됩니다. 그 다음은 원하는 걸 적으면 됩니다.

### 방법 3: /team (팀 작업)

여러 AI가 협력해서 작업합니다.

```
/team 3 타입스크립트 오류 전부 고쳐줘
```

숫자(3)는 몇 명의 AI를 쓸지입니다. 보통 3이면 충분합니다.

### 방법 4: ralph (포기 안 하는 모드)

작업이 완전히 끝날 때까지 절대 포기하지 않습니다.

```
ralph: 로그인 기능 완성해줘
```

앞에 `ralph:` 만 붙이면 됩니다.

---

## 자주 쓰는 명령어 모음

| 하고 싶은 것 | 이렇게 입력하세요 |
|-------------|----------------|
| 뭔가 만들고 싶어 | `autopilot: [원하는 것]` |
| 버그 고치고 싶어 | `ralph: [고칠 내용]` |
| 팀으로 작업하고 싶어 | `/team 3 [할 일]` |
| 계획부터 세우고 싶어 | `/plan [아이디어]` |
| 코드 리뷰 받고 싶어 | `/code-review` |
| 보안 검토 받고 싶어 | `/security-review` |
| 아이디어가 애매할 때 | `/deep-interview "내 아이디어"` |

## 실제 사용 예시

### 예시 1: 웹사이트 만들기

```
autopilot: 간단한 포트폴리오 웹사이트 만들어줘. 내 이름은 홍길동이고 개발자야.
```

→ OMC가 알아서 HTML, CSS, JS 파일 다 만들어줍니다.

### 예시 2: 오류 고치기

```
ralph: 지금 있는 오류 전부 고쳐줘
```

→ 오류가 없어질 때까지 계속 수정합니다.

### 예시 3: 아이디어는 있는데 막막할 때

```
/deep-interview "쇼핑몰 같은 걸 만들고 싶은데 어떻게 해야 할지 모르겠어"
```

→ AI가 질문을 해가면서 아이디어를 구체화해줍니다.

### 예시 4: 코드 이해하기

```
이 코드가 뭐 하는 건지 쉽게 설명해줘
```

→ 그냥 채팅창에 적기만 하면 됩니다.

## 화면 상단 HUD (상태바) 읽는 법

Claude Code를 재시작하면 화면 상단에 이런 게 보입니다:

```
[OMC] branch:main | ctx:45% | todos:2/5
```

| 표시 | 의미 |
|------|------|
| `branch:main` | 현재 작업 중인 브랜치 이름 |
| `ctx:45%` | AI 메모리 사용량 (85% 넘으면 새 세션 시작) |
| `todos:2/5` | 할 일 5개 중 2개 완료 |
| `ralph:3/10` | ralph 모드로 10번 중 3번째 시도 중 |
| `agents:2` | 지금 AI 에이전트 2개가 동시에 작업 중 |

## 자주 묻는 질문

**Q. 영어로 써야 하나요?**
A. 아니요. 한국어로 써도 됩니다. 그냥 말하듯이 적으세요.

**Q. 명령어를 외워야 하나요?**
A. 아니요. `autopilot:` 하나만 기억해도 대부분 됩니다. 나머지는 그냥 한국어로 말하면 돼요.

**Q. autopilot이랑 ralph 차이가 뭔가요?**
A. `autopilot`은 새로운 걸 만들 때, `ralph`는 기존 것을 수정하거나 완성할 때 씁니다.

**Q. /team에서 숫자는 뭔가요?**
A. 동시에 일할 AI 에이전트 수입니다. 복잡할수록 높게, 간단하면 낮게. 보통 3이면 충분합니다.

**Q. 작업 중에 멈추려면?**
A. Claude Code에서 `Esc` 키를 누르거나 `/cancel` 을 입력하세요.

## OMC 업데이트 방법

새 버전이 나오면 이렇게 입력하세요:

```
/plugin marketplace update omc
/omc-setup
```

---

## 문제가 생겼을 때

뭔가 이상하면 이걸 입력하세요:

```
/omc-doctor
```

자동으로 문제를 찾아서 고쳐줍니다.

---

## 핵심 요약

1. **뭔가 만들고 싶다** → `autopilot: [원하는 것]`
2. **뭔가 고치고 싶다** → `ralph: [고칠 것]`
3. **그냥 물어보고 싶다** → 그냥 한국어로 적으면 됨
4. **막막하다** → `/deep-interview "[아이디어]"`
5. **문제 생겼다** → `/omc-doctor`

**이것만 기억하면 충분합니다.**
