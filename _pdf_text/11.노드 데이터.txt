주의사항
캔버스 독립성
데이터 저장소는 각 캔버스마다 독립적으로 운영되며, 다른 캔버스 간에는 데이터 저

노드를 먼저 실행해야 합니다.

설명
텍스트 입력 노드는 서로 다른 종류의 데이터를 각각 입력받을 때 사용합니다.
여러 개의 열(column)을 추가할 수 있어서, 각각 다른 의미를 가진 텍스트를 입력할 수 있습
니다. 예를 들어, 이름, 나이, 주소처럼 서로 다른 정보를 한 번에 입력할 때 유용합니다.

열 추가
