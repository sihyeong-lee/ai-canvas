수평 막대 그래프
위젯 기능
• 수평 막대 그래프로 데이터를 시각화합니다.
연결 방법
• Input: 데이터, 예제데이터
• Output: 
범주형 데이터의 경우-선택된 열의 개별 범주 간의 비교
숫자형 데이터의 경우-14개 구간으로 나누어 구간별 비교

수평 막대 그래프
위젯 설정
•
 캔버스 위젯 안에서 열을 선택할 수 있도록 합니다. 
•
 X축, Y축에 제목을 지정합니다. 
•
 각 데이터의 값을 표시하는 라벨을 설정합니다.
•
 시각화한 데이터 정보를 나타내는 범례의 위치를 선택합니다.
•
 차트 유형을 선택합니다. (없음, 집계)
•
 Numerical의 경우 대상 열에 대하여 범주화하여 평균값을 시각화합니다. 
Categorical의 경우 대상 열에 대하여 범주화하여 값(개수)을 시각화합니다. 
•
 Y축의 여러 설정을 변경합니다. 
•
 Y축을 나타내는 서식을 선택합니다.

있습니다.

• Output: 데이터의 시간 또는 독립 변수에 따른 추세, 패턴
또는 상관 관계를 시각화한 그래프

필터
위젯 기능
• 설정한 조건에 따라 동적 UI를 생성하고 데이터를 필터링합니다.
연결 방법
• Input: 데이터, 예제 데이터 등
• Output: 필터링 된 데이터

필터
위젯 설정
•
 필터 추가하기를 클릭한 후 원하는 조건을 선택합니다. 
•
 필터링할 열을 선택합니다. 
•
• 검색: 직접 데이터를 검색창에 검색합니다. 
• 드롭다운: 클릭하여 나온 여러 데이터 목록 중 하나를 선택합니다. 
• 다중 드롭다운: 클릭하여 나온 여러 데이터 목록 중 여러 데이터를 선택합니다.
• 라디오 선택: 모든 데이터가 필터에 표시되고 그중 한 데이터를 선택합니다. 
• 체크박스: 모든 데이터가 필터에 표시되고 그중 여러 데이터를 선택합니다.
•
• 검색: 직접 데이터를 검색창에 검색합니다. 
• 슬라이더: 슬라이더를 움직여 선택한 값 이상 또는 이하의 값을 모두 필터링합니다. 
•
 필터의 제목을 지정합니다.
